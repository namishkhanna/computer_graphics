In this Computer Graphic code, there is Scaling of an object.

	For Rectangle
1. Specify the first coordinate.
2. Specify the second coordinate.
3. Enter coordinate where you want to scale.

	For Triangle
1. Specify the first coordinate.
2. Specify the second coordinate.
3. Specify the third coordinate.
4. Enter coordinate where you want to scale.

Note: Make sure u have graphic packages installed on your system.
